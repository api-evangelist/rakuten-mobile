# rakuten-mobile

Rakuten Mobile — Japan's cloud-native fourth carrier and the engineering origin of Rakuten Symphony's Symworld platform, CPaaS APIs, and Open RAN portfolio.

## APIs profiled

- **Rakuten CPaaS SMS / Messaging API** — Carrier-grade A2P/P2A SMS with delivery-receipt webhooks. `https://api.cpaas.symphony.rakuten.net/sms/v1`
- **Rakuten CPaaS Confirm (OTP) API** — One-time PIN delivery and verification over SMS and Voice with JWT bearer auth.
- **Rakuten CPaaS Short URL API** — Link shortening, lifecycle management, and click telemetry for CPaaS campaigns.

## Platforms referenced

- **Symworld** — Cloud-native operating platform for telcos: Open RAN (CU/DU/RIC, Symware DU appliance), Symworld Cloud (telco CNP / hyperconverged Kubernetes), OSS/BSS, site management, and a marketplace of 110+ pre-integrated CNFs from 50+ vendor partners.
- **Rakuten Mobile** — Japan's fourth nationwide MNO, 10M+ subscribers (Dec 2025), 99.9% 4G population coverage, expanding 5G plus 700 MHz Platinum Band.
